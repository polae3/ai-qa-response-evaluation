# AI 생성 응답 품질 평가 및 오류 패턴 분석

## 프로젝트 개요
이 프로젝트는 AI가 생성한 응답 데이터를 대상으로 품질을 평가하고, 오류 유형을 분류하여 반복되는 문제 패턴을 분석한 포트폴리오입니다.

AI QA Assistant 직무와 관련하여 다음 역량을 보여주는 것을 목표로 했습니다.

- AI 응답 품질 평가
- 오류 사례 수집 및 유형 분류
- 반복 오류 패턴 분석
- 데이터 기반 개선 포인트 도출

## 사용 기술
- Python
- pandas
- Jupyter Notebook

## 데이터 구성
데이터셋은 다음 컬럼으로 구성했습니다.

- `question`: 사용자 질문
- `category`: 질문 유형
- `ai_response`: AI 응답
- `score`: 응답 품질 점수 (1~5)
- `error_type`: 오류 유형
- `comment`: 평가 코멘트

## 오류 유형 정의
- `none`: 오류 없음
- `minor`: 경미한 문제
- `partial`: 부분 정답 / 설명 부족
- `wrong`: 명백한 사실 오류 또는 부정확한 응답

## 분석 내용
- 전체 평균 점수 확인
- 오류 유형 분포 분석
- 카테고리별 평균 점수 비교
- 저품질 응답(score <= 3) 추출
- 응답 길이와 점수의 관계 확인
- 오류 유형별 샘플 검토

## 주요 결과
- 일부 fact / comparison / definition 유형에서 품질 저하가 발생했다.
- `wrong` 유형은 사실 오류, `partial` 유형은 설명 부족에서 주로 발생했다.
- 반복 오류를 기준으로 QA 가이드라인을 개선할 수 있음을 확인했다.

## 파일 구성
- `dataset.csv`: 평가용 샘플 데이터
- `ai_response_analysis.ipynb`: 분석 노트북
- `README.md`: 프로젝트 설명 문서

## 직무 연관성
본 프로젝트는 AI가 생성한 결과물을 평가하고, 품질 문제를 구조화하여 개선 포인트를 제시하는 과정에 초점을 두었습니다.  
이는 AI QA Assistant 직무의 핵심 업무인 응답 검수, 오류 분석, 패턴 정리, 품질 개선과 직접적으로 연결됩니다.
